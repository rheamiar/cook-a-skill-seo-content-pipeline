# Skill Card — SEO Content Pipeline

| Mục | Nội dung |
|---|---|
| **Tên Skill** | SEO Content Pipeline |
| **Việc gì được automate** | Toàn bộ quy trình tạo nội dung SEO từ đầu đến cuối trong một pipeline liên tục: <br>• Research keyword từ 3 nguồn dữ liệu thật <br>• Lên outline theo logic flow phù hợp search intent <br>• Viết draft human-style có data và nguồn cụ thể <br>• Tối ưu on-page SEO + GEO trong cùng một checklist <br>• Detect và rewrite AI writing patterns <br>• Chấm điểm 6 tiêu chí và đưa ra actionable fixes |
| **TRƯỚC: làm tay** | 4–5 tiếng/bài, chất lượng SEO không nhất quán: <br>• Keyword research thủ công trên Ahrefs, Google Trends xem riêng, social buzz đoán mò <br>• Outline tự nghĩ, không có framework chuẩn <br>• Viết draft là bước tốn thời gian nhất (2–3 tiếng) <br>• Optimize SEO sau khi viết xong bằng cách check thủ công từng tiêu chí <br>• Không có bước chấm điểm — không biết bài đã đủ chuẩn chưa trước khi publish |
| **SAU: có skill** | 15–20 phút/bài, output có score và ready-to-fix: <br>• Keyword research tự động từ 3 nguồn với scoring formula có trọng số <br>• Outline generate theo đúng flow pattern phù hợp search intent <br>• Draft viết đúng chuẩn human-style, có tối thiểu 3 data point có nguồn thật <br>• SEO score + GEO score + priority fixes đi kèm mỗi output <br>• Không cần chuyển tab hay tool trong suốt quá trình |
| **Tool/AI đã dùng** | • Claude (claude.ai Projects) <br>• Ahrefs API <br>• Semrush Free Keyword Tool <br>• Google Trends (via web search) <br>• X/Twitter (via web search) |
| **Limitation** | • Ahrefs API cần trả phí — free mode dùng estimated data, không chính xác bằng <br>• Social data từ X/Twitter là estimate qua web search, không phải live API <br>• Tối ưu cho content dạng informational và commercial investigation — chưa phù hợp cho landing page hoặc product page <br>• Chưa connect với CMS để auto-publish |
| **Roadmap mở rộng** | • Thêm competitor content gap analysis (Stage 1.5) <br>• Hỗ trợ tiếng Việt với SEO rules riêng cho thị trường VN <br>• Internal linking suggestions dựa trên content library có sẵn <br>• Connect Google Search Console để lấy real ranking data làm input <br>• Thêm checkpoint approval giữa Stage 2 và 3 để editor review outline trước khi viết draft <br>• CMS integration (WordPress/Ghost) để publish thẳng từ output |
