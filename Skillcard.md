# Skill Card — SEO Content Pipeline

| Mục | Nội dung |
|---|---|
| **Tên Skill** | SEO Content Pipeline |
| **Việc gì được automate** | Toàn bộ quy trình tạo nội dung SEO: từ research keyword, lên outline logic, viết draft human-style, tối ưu on-page SEO + GEO, detect AI patterns, đến chấm điểm và đưa ra actionable fixes — trong một pipeline liên tục, không cần chuyển tab hay tool |
| **TRƯỚC: làm tay** | 4–5 tiếng/bài. Keyword research thủ công trên Ahrefs, Google Trends xem riêng, social buzz đoán mò. Outline tự nghĩ, không có framework. Viết draft là bước tốn thời gian nhất (2–3 tiếng). Optimize SEO sau khi viết xong bằng cách check thủ công từng tiêu chí. Không có bước chấm điểm — chất lượng SEO không nhất quán giữa các bài. |
| **SAU: có skill** | 15–20 phút/bài. Keyword research tự động từ 3 nguồn (Ahrefs, Google Trends, X/Twitter) với scoring formula có trọng số. Outline được generate theo logic flow pattern phù hợp search intent. Draft viết theo đúng chuẩn human-style với data và nguồn thật. Output kèm SEO score + GEO score + priority fixes sẵn sàng để publish. |
| **Tool/AI đã dùng** | Claude (claude.ai Projects) · Ahrefs API · Semrush Free Keyword Tool · Google Trends (via web search) · X/Twitter (via web search) |
| **Limitation** | Ahrefs API cần trả phí — free mode dùng estimated data, không chính xác bằng. Social data từ X/Twitter là estimate qua web search, không phải live API. Skill tối ưu cho content dạng informational và commercial investigation — chưa phù hợp cho landing page hoặc product page. Chưa connect với CMS để auto-publish. |
| **Roadmap mở rộng** | Thêm competitor content gap analysis (Stage 1.5) · Hỗ trợ tiếng Việt với SEO rules riêng cho thị trường VN · Internal linking suggestions dựa trên content library có sẵn · Connect Google Search Console để lấy real ranking data làm input · Thêm checkpoint approval giữa Stage 2 và Stage 3 để editor review outline trước khi viết draft · CMS integration (WordPress/Ghost) để publish thẳng từ output |
