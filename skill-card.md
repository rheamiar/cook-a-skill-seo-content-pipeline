# Skill Card — SEO Content Pipeline

| Mục | Nội dung |
|---|---|
| **Tên Skill** | SEO Content Pipeline |
| **Việc gì được automate** | Toàn bộ quy trình tạo nội dung SEO từ đầu đến cuối trong một pipeline liên tục: <br>• Research keyword từ 3 nguồn data thực (Ahrefs, Google Trends, X/Twitter) <br>• Lên outline logic theo flow pattern phù hợp search intent <br>• Viết draft human-style với data và nguồn dẫn thật <br>• Tối ưu on-page SEO + GEO optimization <br>• Detect và rewrite AI writing patterns <br>• Chấm điểm 6 chiều kèm priority fixes sẵn sàng để publish |
| **TRƯỚC: làm tay** | 4–5 tiếng/bài, chất lượng SEO không nhất quán: <br>• Keyword research thủ công trên từng tool riêng lẻ <br>• Outline tự nghĩ, không có framework chuẩn <br>• Viết draft là bottleneck lớn nhất (2–3 tiếng) <br>• Optimize SEO sau khi viết xong bằng cách check thủ công từng tiêu chí <br>• Không có bước chấm điểm — không biết bài có sẵn sàng để publish chưa |
| **SAU: có skill** | 15–20 phút/bài, output kèm score và ready-to-fix notes: <br>• Keyword research tự động từ 3 nguồn với weighted scoring formula <br>• Outline generate theo đúng flow pattern phù hợp search intent <br>• Draft viết chuẩn human-style với tối thiểu 3 data points có nguồn thật <br>• SEO score + GEO score + priority fixes có trong mỗi output <br>• Không cần chuyển tab hay tool trong suốt toàn bộ quy trình |
| **Tool / AI đã dùng** | • Claude (claude.ai Projects) <br>• Ahrefs API <br>• Google Trends (qua web search) <br>• X/Twitter (qua web search) <br>• Semrush Free Keyword Tool (fallback) |
| **Limitation** | • Ahrefs API cần trả phí — free mode chỉ tìm keyword candidates, không có volume data thật <br>• Social data từ X/Twitter là estimate qua web search, không phải live API <br>• Tối ưu cho content dạng informational và commercial investigation — chưa phù hợp cho landing page hoặc product page <br>• Chưa connect với CMS để auto-publish <br>• Claude đôi khi vẫn dùng em dash trong output dù đã có rule — nên review thủ công trước khi publish |
| **Roadmap mở rộng** | • Competitor content gap analysis (Stage 1.5) <br>• Hỗ trợ tiếng Việt với SEO rules riêng cho thị trường VN <br>• Internal linking suggestions dựa trên content library có sẵn <br>• Google Search Console integration để lấy real ranking data làm input <br>• Multiple entry points: optimize draft có sẵn (Stage 4→5→6) hoặc rewrite từ outline (Stage 2→3→4→5→6) <br>• CMS integration (WordPress/Ghost) để publish thẳng từ output <br>• Native Ahrefs API integration qua Claude Code để pull live data không cần copy-paste thủ công |
